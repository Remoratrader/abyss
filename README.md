# Abyss Explorer
A 3D interactive visualization of the layers of the internet and programming languages.

## Concept
- **Surface Web**: Public, indexed layers.
- **Deep Web**: Private databases and restricted content.
- **Dark Web**: Anonymized and encrypted networks.
- **The Abyss**: Classified systems and experimental codes.

## Tech Stack
- Three.js
- Vite
- Vanilla JS/CSS

## Controls
- Click a zone to analyze layer details.
- Drag to orbit around the diorama.
- Scroll to zoom.
- [📷 ROV Cam] to enter first-person scanning mode.
- [WASD] to navigate in ROV mode.
- [ESC] to exit analysis or ROV mode.
